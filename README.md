# Real-Time Chat Application (Node.js + Socket.IO)

A simple and efficient **real-time chat application** built using **Node.js, Express and Socket.IO**.
This application enables multiple users to communicate instantly through WebSockets without using a database.

---

## Features

✅ Real-time bi-directional messaging
✅ Multiple users support
✅ Join & leave notifications
✅ Clean and responsive UI
✅ No database required
✅ Built using WebSocket (Socket.IO)

---

## Tech Stack

* Node.js
* Express.js
* Socket.IO
* HTML
* CSS
* JavaScript

---

## How to Run This Project Locally

1. Clone the repository:

   ```
   git clone https://github.com/<your-username>/real-time-chat-app.git
   ```

2. Go inside the project folder:

   ```
   cd real-time-chat-app
   ```

3. Install required packages:

   ```
   npm install
   ```

4. Start the server:

   ```
   node server.js
   ```

5. Open in your browser:

   ```
   http://localhost:3000
   ```

## How It Works

* The server is built using **Node.js + Express**
* **Socket.IO** is used for real-time communication
* Messages are broadcasted to all connected clients instantly
* No database is needed — messages exist only during the session
